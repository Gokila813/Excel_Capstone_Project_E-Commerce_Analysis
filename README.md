# E-Commerce Sales Analysis using Excel

## Project Overview
This project analyzes e-commerce sales data using Microsoft Excel to identify sales trends, customer behavior, and business insights.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Conditional Formatting

## Key Analysis
- Monthly Sales Trends
- Top-Selling Products
- Store-wise analysis
- Top-Customer by sales
- Gender-wise analysis

## Features
- Interactive Dashboard
- Data Cleaning
- Sales Visualization
- Business Reporting
